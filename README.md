# wireui

A tool for creating and managing wireguard configs  
Version 0.1.0b1  
&copy; 2020-2021 Tim Schlottmann  

## Installation

Please run
```
python3 -m pip install wireui-TheTimmoth
```

## Usage

Please run
```
python3 -m wireui
```
